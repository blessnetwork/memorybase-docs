---
title: "Context Packs"
description: "A portable summary of everything in a project, ready to use in any AI tool."
---

A context pack is a single document that summarizes everything in a project -- conversations, notes, and sources. Instead of re-explaining your project to a new AI session, you can share the context pack and it's instantly up to speed.

## How they work

When you add conversations to a project, MemoryBase automatically extracts the key information:

- What was discussed
- Decisions that were made
- Important terms and concepts

This is assembled into a context pack that updates as you add more conversations.

## Using a context pack

Open a project and click **Context pack** to view it. Click **Export as Markdown** to download it. You can then paste it into any AI tool's prompt, upload it as a file, or use it as a reference.
