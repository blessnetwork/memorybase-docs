---
title: "Timeline"
description: "AI-generated daily summaries of your conversation activity."
---

The timeline gives you a daily view of your AI conversation activity. Each entry is an LLM-generated summary of that day's conversations, with extracted topics, tags, and links back to the source conversations.

Entries are generated automatically after conversations sync. No manual action required.

## Entry fields

| Field | Description |
|-------|-------------|
| `date` | Calendar day covered |
| `summary` | AI-generated activity summary |
| `tags` / `topics` | Extracted keywords and themes |
| `tagMappings` | Maps tags to the conversation IDs where they appeared |
| `conversations` | Source conversations with title, platform, tags, topics, summary |
| `status` | `completed`, `processing`, or `not_generated` |

## Tag mappings

Tags link back to specific conversations:

```json
{
  "tagMappings": {
    "typescript": ["conv-1"],
    "debugging": ["conv-1", "conv-2"]
  }
}
```

## Manual generation

If an entry shows `not_generated`, you can trigger generation manually. You can also regenerate existing entries to incorporate newly synced conversations.
