---
title: "Export, Import & Wipe"
description: "Back up your data as an encrypted file, import it anywhere, or wipe local storage completely."
---

## Export

Exports all profiles, memories, and settings into a single encrypted file with integrity checksum.

**Settings > Data > Export** — set a password, click Export. No recovery if you lose the password.

## Import

**Settings > Data > Import** — select file, enter password, choose mode:

- **Replace** — overwrites all local data with the import file.
- **Merge** — combines imported data with existing data. ID conflicts resolved automatically.

Import validates file integrity first. Corrupted files or wrong passwords are rejected without touching existing data.

## Wipe

**Settings > Data > Wipe** — removes all local profiles, memories, and settings. Immediate and permanent.

Export first. Type `wipe` to confirm.

<Warning>
  Cloud-synced data must be deleted separately from your account settings.
</Warning>

## Storage usage

The extension popup shows a storage bar at the bottom.

| Level | Meaning |
|-------|---------|
| Below 80% | Normal |
| 80%+ | Warning — consider freeing space |
| 95%+ | Critical — auto-save paused |
