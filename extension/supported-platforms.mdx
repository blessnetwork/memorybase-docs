title: "Supported Platforms"
description: "Platforms Memorybase integrates with: ChatGPT, Claude, Gemini, and manual fallback."
---

Memorybase uses adapters to integrate with AI platforms. Each adapter provides conversation capture, toolbar injection, message save buttons, and prompt injection.

## Platforms

| Platform | URL | Status |
|----------|-----|--------|
| **ChatGPT** | chatgpt.com | Full integration |
| **Claude** | claude.ai | Full integration |
| **Gemini** | gemini.google.com | Partial — actively developing |
| **Any site** | Universal fallback | Right-click to save text |

## Adapter health

| Status | Meaning |
|--------|---------|
| **Healthy** | All features working |
| **Degraded** | Some features broken, fallbacks active |
| **Broken** | Manual mode only |

Check adapter status in the popup under **Integrations**.

<Warning>
  AI platforms change their UI without notice. When an adapter breaks, Memorybase falls back to manual mode. Fixes ship as patch updates.
</Warning>

## Manual fallback

Works everywhere: select text on any page, right-click, choose **Save to Memorybase**. Works on PDFs, docs, research papers — not just chat interfaces.
