The Memorybase Chrome extension captures your AI conversations and lets you inject stored memories into prompts — directly inside ChatGPT, Claude, and Gemini.

## Icon states

| Icon | Meaning |
|------|---------|
| Grey | Not active on this site |
| Green | Recording conversations |
| Amber (`||`) | Paused |
| Red (`!`) | Error — action needed |

## Popup

Click the toolbar icon to open the popup. Three sections:

- **Memory Injection** — pause/resume injection with configurable duration.
- **My Memory** — memory count and link to your dashboard.
- **Integrations** — platform status, last sync, manual refresh.

## Overlay

On supported platforms, a button appears near the chat input showing memory status: *ready*, *syncing*, *attached (N)*, or *paused*. Click it to browse and attach memories.

## Keyboard shortcuts

| Action | Shortcut |
|--------|----------|
| Toggle pause | `Alt+Shift+P` |
| Quick profile switch | `Ctrl+Shift+P` / `Cmd+Shift+P` |
| Improve prompt | `Ctrl+Shift+I` / `Cmd+Shift+I` |
| Add context | `Ctrl+Shift+C` / `Cmd+Shift+C` |
