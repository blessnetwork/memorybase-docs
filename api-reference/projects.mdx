---
title: "Projects — Memorybase API Reference"
description: "Create and manage projects to organise your conversations, attach notes, URLs, and snippets as sources, and export assembled context packs."
---

Projects let you group related conversations and attach your own notes, URLs, and code snippets as sources. Memorybase automatically assembles a **context pack** from each project — a curated summary you can inject into any AI session. Two special system projects exist for every account: **Uncategorized** (conversations not yet assigned to a project) and **Archived** (conversations you have archived).

---

### GET /projects

List all projects for your account, each including their sources and linked conversations.

**Example request**

```bash
curl "https://api.memorybase.app/projects" \
  -H "Authorization: Bearer <token>"
```

**Example response**

```json
[
  {
    "id": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
    "userId": "user-abc123",
    "name": "React + TypeScript App",
    "description": "Tracking conversations about building a React application with TypeScript",
    "isArchived": false,
    "version": 1,
    "createdAt": "2026-02-25T10:00:00.000Z",
    "updatedAt": "2026-02-25T10:00:00.000Z",
    "deletedAt": null,
    "sources": [],
    "conversations": []
  }
]
```

### POST /projects

Create a new project.

**Body**

<ParamField body="name" type="string" required>
  Project name. Minimum 1 character, maximum 200 characters.
</ParamField>

<ParamField body="description" type="string">
  Optional description. Maximum 1000 characters.
</ParamField>

**Example request**

```bash
curl -X POST "https://api.memorybase.app/projects" \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "React + TypeScript App",
    "description": "Tracking conversations about building a React application with TypeScript"
  }'
```

**Example response** — `201 Created`

```json
{
  "id": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
  "userId": "user-abc123",
  "name": "React + TypeScript App",
  "description": "Tracking conversations about building a React application with TypeScript",
  "isArchived": false,
  "version": 1,
  "createdAt": "2026-02-25T10:00:00.000Z",
  "updatedAt": "2026-02-25T10:00:00.000Z",
  "deletedAt": null
}
```

### PUT /projects/:id

Update a project's name, description, or archived state.

**Path parameters**

<ParamField path="id" type="string" required>
  UUID of the project.
</ParamField>

**Body**

<ParamField body="name" type="string">
  New name. Minimum 1 character, maximum 200 characters.
</ParamField>

<ParamField body="description" type="string">
  New description. Maximum 1000 characters.
</ParamField>

<ParamField body="isArchived" type="boolean">
  Set to `true` to archive the project, `false` to restore it.
</ParamField>

**Example request**

```bash
curl -X PUT "https://api.memorybase.app/projects/a1b2c3d4-e5f6-7890-abcd-ef1234567890" \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "React + TypeScript App (Updated)",
    "description": "Updated project description",
    "isArchived": false
  }'
```

Returns the updated project base object.

### DELETE /projects/:id

Permanently delete a project. This does not delete the conversations linked to the project.

**Path parameters**

<ParamField path="id" type="string" required>
  UUID of the project to delete.
</ParamField>

**Example request**

```bash
curl -X DELETE "https://api.memorybase.app/projects/a1b2c3d4-e5f6-7890-abcd-ef1234567890" \
  -H "Authorization: Bearer <token>"
```

Returns `204 No Content` on success.

### POST /projects/:id/sources

Add a source (note, URL, or code snippet) to a project.

**Path parameters**

<ParamField path="id" type="string" required>
  UUID of the project.
</ParamField>

**Body**

<ParamField body="content" type="string" required>
  The source content. For `url` type this is the URL string; for `note` and `snippet` types this is the text content. Minimum 1 character.
</ParamField>

<ParamField body="type" type="string" required>
  Source type. One of `note`, `url`, or `snippet`.
</ParamField>

<ParamField body="title" type="string">
  Optional display title for the source. Minimum 1 character if provided.
</ParamField>

**Example request**

```bash
curl -X POST "https://api.memorybase.app/projects/a1b2c3d4-e5f6-7890-abcd-ef1234567890/sources" \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "TypeScript Docs",
    "content": "https://react.dev/learn/typescript",
    "type": "url"
  }'
```

**Example response** — `201 Created`

```json
{
  "id": "s1a2b3c4-d5e6-7890-abcd-ef1234567890",
  "projectId": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
  "userId": "user-abc123",
  "title": "TypeScript Docs",
  "content": "https://react.dev/learn/typescript",
  "type": "url",
  "createdAt": "2026-02-25T10:00:00.000Z",
  "updatedAt": "2026-02-25T10:00:00.000Z"
}
```

### GET /projects/:id/context-pack

Retrieve the assembled context pack for a project. The context pack aggregates AI-extracted insights from every linked conversation into a single, structured object.

**Path parameters**

<ParamField path="id" type="string" required>
  UUID of the project.
</ParamField>

**Example request**

```bash
curl "https://api.memorybase.app/projects/a1b2c3d4-e5f6-7890-abcd-ef1234567890/context-pack" \
  -H "Authorization: Bearer <token>"
```

**Example response**

```json
{
  "projectName": "React + TypeScript App",
  "description": "Tracking conversations about building a React application with TypeScript",
  "topics": ["react", "typescript"],
  "tags": ["frontend", "hooks"],
  "layers": [],
  "version": 1,
  "layerCount": 5,
  "completedCount": 4,
  "pendingCount": 1,
  "failedCount": 0
}
```

**Response fields**

<ResponseField name="projectName" type="string">Display name of the project.</ResponseField>
<ResponseField name="description" type="string | null">Project description.</ResponseField>
<ResponseField name="topics" type="string[]">Aggregated topics across all conversations.</ResponseField>
<ResponseField name="tags" type="string[]">Aggregated tags across all conversations.</ResponseField>
<ResponseField name="layers" type="array">Per-conversation context layers, each containing a `summary`, `tags`, `topics`, and `contexts` array.</ResponseField>
<ResponseField name="version" type="number">Monotonically increasing version number for this context pack.</ResponseField>
<ResponseField name="layerCount" type="number">Total number of layers (one per linked conversation).</ResponseField>
<ResponseField name="completedCount" type="number">Number of layers that have been fully processed.</ResponseField>
<ResponseField name="pendingCount" type="number">Number of layers still being processed.</ResponseField>
<ResponseField name="failedCount" type="number">Number of layers that failed to process.</ResponseField>

### GET /projects/:id/context-pack/export

Export the project's context pack as a Markdown document, ready to paste directly into any AI system prompt.

**Path parameters**

<ParamField path="id" type="string" required>
  UUID of the project.
</ParamField>

<Note>
  This endpoint returns `Content-Type: text/markdown; charset=utf-8`, not JSON. Use the raw response body as your Markdown text.
</Note>

**Example request**

```bash
curl "https://api.memorybase.app/projects/a1b2c3d4-e5f6-7890-abcd-ef1234567890/context-pack/export" \
  -H "Authorization: Bearer <token>"
```

Returns a Markdown string on success, or a `404` JSON error if the context pack is not found.
