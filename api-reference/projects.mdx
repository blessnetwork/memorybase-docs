---
title: "Projects"
description: "Create and manage projects to organize conversations, attach sources, and export context packs."
---

Projects group related conversations and let you attach notes, URLs, and code snippets as sources. Each project has an auto-assembled **context pack** you can inject into any AI session.

Two system projects exist for every account: **Uncategorized** and **Archived**.

---

### GET /projects

List all projects with their sources and linked conversations.

```bash
curl "https://api.memorybase.app/projects" \
  -H "Authorization: Bearer <token>"
```

---

### POST /projects

Create a new project.

<ParamField body="name" type="string" required>Project name (1-200 chars).</ParamField>
<ParamField body="description" type="string">Optional description (max 1000 chars).</ParamField>

```bash
curl -X POST "https://api.memorybase.app/projects" \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{"name": "React + TypeScript App", "description": "Tracking React/TS conversations"}'
```

Returns `201 Created` with the project object.

---

### GET /projects/suggestions

AI-generated project suggestions based on unorganized conversations.

```bash
curl "https://api.memorybase.app/projects/suggestions" \
  -H "Authorization: Bearer <token>"
```

Returns `{ "suggestions": [{ "name", "description", "conversationIds", "conversations" }] }`.

### GET /projects/:id

Retrieve a project by UUID with sources and conversations.

<ParamField path="id" type="string" required>Project UUID.</ParamField>

```bash
curl "https://api.memorybase.app/projects/a1b2c3d4-e5f6-7890-abcd-ef1234567890" \
  -H "Authorization: Bearer <token>"
```

Returns `404` if not found.

---

### PUT /projects/:id

Update a project.

<ParamField path="id" type="string" required>Project UUID.</ParamField>
<ParamField body="name" type="string">New name (1-200 chars).</ParamField>
<ParamField body="description" type="string">New description (max 1000 chars).</ParamField>
<ParamField body="isArchived" type="boolean">Archive or restore.</ParamField>

```bash
curl -X PUT "https://api.memorybase.app/projects/a1b2c3d4-e5f6-7890-abcd-ef1234567890" \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{"name": "Updated Name", "isArchived": false}'
```

---

### DELETE /projects/:id

Permanently delete a project. Linked conversations are not deleted.

<ParamField path="id" type="string" required>Project UUID.</ParamField>

```bash
curl -X DELETE "https://api.memorybase.app/projects/a1b2c3d4-e5f6-7890-abcd-ef1234567890" \
  -H "Authorization: Bearer <token>"
```

Returns `204 No Content`.

---

### POST /projects/:id/sources

Add a source to a project.

<ParamField path="id" type="string" required>Project UUID.</ParamField>
<ParamField body="content" type="string" required>Source content (URL string, note text, or code snippet).</ParamField>
<ParamField body="type" type="string" required>One of `note`, `url`, or `snippet`.</ParamField>
<ParamField body="title" type="string">Optional display title.</ParamField>

```bash
curl -X POST "https://api.memorybase.app/projects/a1b2c3d4-e5f6-7890-abcd-ef1234567890/sources" \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{"title": "TypeScript Docs", "content": "https://react.dev/learn/typescript", "type": "url"}'
```

Returns `201 Created` with the source object.

### POST /projects/:id/conversations

Link conversations to a project.

<ParamField path="id" type="string" required>Project UUID.</ParamField>
<ParamField body="conversationIds" type="string[]" required>Conversation UUIDs (1-100).</ParamField>

```bash
curl -X POST "https://api.memorybase.app/projects/a1b2c3d4-e5f6-7890-abcd-ef1234567890/conversations" \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{"conversationIds": ["conv-uuid-1", "conv-uuid-2"]}'
```

Returns `{ "success": true }`.
