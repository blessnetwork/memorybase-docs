---
title: "Search"
description: "Full-text search across projects and conversations."
---

Search returns a discriminated union of project and conversation results. Empty query returns recent items as suggestions.

---

### GET /search

<ParamField query="q" type="string">Search query (max 500 chars). Supports `type:projects` or `type:conversations` prefix. Omit for recent suggestions.</ParamField>
<ParamField query="type" type="string">Filter to `projects` or `conversations`. Overrides `type:` prefix in `q`.</ParamField>
<ParamField query="limit" type="number">Max results per type (max `100`, default `5`).</ParamField>
<ParamField query="offset" type="number">Skip results for pagination (default `0`).</ParamField>

```bash
curl "https://api.memorybase.app/search?q=react&limit=20" \
  -H "Authorization: Bearer <token>"
```

```bash
# Filter to conversations only
curl "https://api.memorybase.app/search?q=react&type=conversations&limit=20" \
  -H "Authorization: Bearer <token>"
```

**Response**

```json
{
  "results": [
    {
      "type": "project",
      "data": {
        "id": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
        "name": "React + TypeScript App",
        "description": "Tracking conversations about building a React application",
        "isArchived": false
      }
    },
    {
      "type": "conversation",
      "data": {
        "id": "conv-abc123",
        "platform": "chatgpt",
        "title": "React hooks deep dive",
        "isArchived": false,
        "metadata": {
          "tags": ["react", "hooks"],
          "topics": ["frontend"],
          "summary": "Discussion about React hooks patterns"
        }
      }
    }
  ],
  "total": 2,
  "limit": 20,
  "offset": 0,
  "filter": null,
  "query": "react"
}
```

Each result has `type` (`"project"` or `"conversation"`) and `data` with the full resource object. Response includes `total`, `limit`, `offset`, `filter`, and cleaned `query`.
