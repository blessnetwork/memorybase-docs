title: "Settings"
description: "Profile, linked accounts, and avatar management."
---

### GET /settings/profile

Get your profile.

```bash
curl "https://api.memorybase.app/settings/profile" \
  -H "Authorization: Bearer <token>"
```

```json
{
  "id": "user-abc123",
  "email": "user@example.com",
  "name": "Jane Doe",
  "emailVerified": true,
  "image": null,
  "tier": "pro",
  "createdAt": "2026-02-25T10:00:00.000Z"
}
```

---

### PATCH /settings/profile

Update your display name.

<ParamField body="name" type="string" required>New name (1-100 chars).</ParamField>

```bash
curl -X PATCH "https://api.memorybase.app/settings/profile" \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{"name": "Jane Doe"}'
```

Returns the updated profile object.

---

### GET /settings/accounts

List linked authentication accounts (email OTP + any OAuth providers).

```bash
curl "https://api.memorybase.app/settings/accounts" \
  -H "Authorization: Bearer <token>"
```

```json
[
  {
    "id": "acct-abc123",
    "providerId": "email-otp",
    "accountId": "user@example.com",
    "createdAt": "2026-02-25T10:00:00.000Z"
  }
]
```

---

### DELETE /settings/accounts/:accountId

Unlink an authentication account.

<ParamField path="accountId" type="string" required>Account ID from `GET /settings/accounts`.</ParamField>

```bash
curl -X DELETE "https://api.memorybase.app/settings/accounts/acct-abc123" \
  -H "Authorization: Bearer <token>"
```

Returns `204 No Content`. Cannot unlink email OTP (`email-*`) or your only remaining account.

---

### PUT /settings/profile/avatar

Upload a new avatar. Send as `multipart/form-data`. Accepts `image/jpeg`, `image/png`, `image/webp`, `image/gif`. Max 2 MB.

```bash
curl -X PUT "https://api.memorybase.app/settings/profile/avatar" \
  -H "Authorization: Bearer <token>" \
  -F "file=@/path/to/avatar.png"
```

Returns `{ "image": "https://avatars.memorybase.io/..." }`.

---

### DELETE /settings/profile/avatar

Remove your avatar.

```bash
curl -X DELETE "https://api.memorybase.app/settings/profile/avatar" \
  -H "Authorization: Bearer <token>"
```

Returns `204 No Content`.
