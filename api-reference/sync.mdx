---
title: "Sync"
description: "Low-level sync endpoints used by the Memorybase extension. For advanced integrations only."
---

These endpoints are the transport layer the browser extension uses internally. For normal use, prefer the [Conversations](/api-reference/conversations) and [Projects](/api-reference/projects) APIs.

---

### POST /sync/incremental

Push a batch of create/update/delete operations.

<ParamField body="lastSyncedAt" type="string | null" required>ISO 8601 timestamp of last sync, or `null` for first sync.</ParamField>
<ParamField body="operations" type="array" required>Array of operations, each with `id`, `resourceType` (`conversation`|`message`), `resourceId`, `operation` (`create`|`update`|`delete`), optional `data`, and `timestamp`.</ParamField>
<ParamField body="browserCache" type="array">Optional browser cache entries with `clientId`, `url`, `domain`, `title`, `favIconUrl`, `visitedAt`.</ParamField>

```bash
curl -X POST "https://api.memorybase.app/sync/incremental" \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{
    "lastSyncedAt": "2026-02-25T09:00:00.000Z",
    "operations": [{
      "id": "op-001",
      "resourceType": "conversation",
      "resourceId": "conv-123",
      "operation": "create",
      "data": {"platform": "claude", "title": "My conversation"},
      "timestamp": "2026-02-25T10:30:00.000Z"
    }]
  }'
```

Returns `state` (`success`|`partial`|`failed`), `syncedAt`, and `results` broken down by resource type.

### GET /sync/status

Get server-side record counts.

```bash
curl "https://api.memorybase.app/sync/status" \
  -H "Authorization: Bearer <token>"
```

```json
{
  "userId": "user-abc123",
  "conversationCount": 142,
  "messageCount": 2847
}
```

### POST /sync/verify

Check which IDs are missing on the server.

<ParamField body="conversations" type="string[]" required>Conversation IDs to verify (max 1000).</ParamField>
<ParamField body="messages" type="string[]" required>Message IDs to verify (max 1000).</ParamField>

```bash
curl -X POST "https://api.memorybase.app/sync/verify" \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{"conversations": ["conv-123"], "messages": ["msg-456"]}'
```

Returns `{ "missingConversations": [], "missingMessages": ["msg-456"] }`.

### POST /sync/timelines

Pull changed timeline entries by comparing local fingerprints.

<ParamField body="versions" type="object" required>Map of `YYYY-MM-DD` date strings to fingerprint strings.</ParamField>

```bash
curl -X POST "https://api.memorybase.app/sync/timelines" \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{"versions": {"2026-03-25": "1742000000000"}}'
```

Returns `{ "timelines": [...], "deleted": [] }`.

---

### POST /sync/context-packs

Pull changed context packs by comparing local fingerprints.

<ParamField body="versions" type="object" required>Map of project IDs to fingerprint strings.</ParamField>

```bash
curl -X POST "https://api.memorybase.app/sync/context-packs" \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{"versions": {"proj-abc123": "3_1740000000000"}}'
```

Returns `{ "packs": [...], "deleted": [] }`.
