---
title: "Conversations"
description: "List, retrieve, and manage AI conversations captured by Memorybase."
---

Conversations are threads captured from supported AI platforms (Claude, ChatGPT, Gemini).

---

### GET /conversations

List conversations, ordered by most recently updated.

<ParamField query="ids" type="string">Comma-separated UUIDs (max 100).</ParamField>
<ParamField query="platform" type="string">Filter by platform: `claude`, `chatgpt`, `gemini`.</ParamField>
<ParamField query="cursor" type="string">Pagination cursor from `nextCursor`.</ParamField>
<ParamField query="limit" type="number">Max results (max `100`).</ParamField>

```bash
curl "https://api.memorybase.app/conversations?platform=claude&limit=20" \
  -H "Authorization: Bearer <token>"
```

```json
{
  "conversations": [
    {
      "id": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",

      "updatedAt": "2026-02-25T09:45:00.000Z",
      "metadata": {
        "tags": ["typescript", "generics"],
        "topics": ["type-safety", "debugging"],
        "summary": "Discussed how to debug complex TypeScript generic constraints.",
        "userTags": [],
        "userTopics": [],
        "userSummary": null,
        "status": "completed",
        "messageCount": 24
      },
      "projects": []
    }
  ],
  "nextCursor": null
}
```

### GET /conversations/:id

Retrieve a single conversation by UUID.

<ParamField path="id" type="string" required>Conversation UUID.</ParamField>

```bash
curl "https://api.memorybase.app/conversations/a1b2c3d4-e5f6-7890-abcd-ef1234567890" \
  -H "Authorization: Bearer <token>"
```

Returns `404` if not found.

### POST /conversations/:id/projects

Replace the full set of linked projects for a conversation.

<ParamField path="id" type="string" required>Conversation UUID.</ParamField>
<ParamField body="projectIds" type="string[]" required>Project UUIDs to link (max 100). Empty array removes all links.</ParamField>

```bash
curl -X POST "https://api.memorybase.app/conversations/a1b2c3d4-e5f6-7890-abcd-ef1234567890/projects" \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{"projectIds": ["d4e5f6a7-b8c9-0123-defa-234567890123"]}'
```

Returns `{ "success": true }`. Returns `404` if conversation or any project not found.

### DELETE /conversations/:id/projects/:projectId

Unlink a single project from a conversation.

<ParamField path="id" type="string" required>Conversation UUID.</ParamField>
<ParamField path="projectId" type="string" required>Project UUID.</ParamField>

```bash
curl -X DELETE \
  "https://api.memorybase.app/conversations/a1b2c3d4-e5f6-7890-abcd-ef1234567890/projects/d4e5f6a7-b8c9-0123-defa-234567890123" \
  -H "Authorization: Bearer <token>"
```

Returns `204 No Content`. Returns `404` if not found.

---

### POST /conversations/:id/archive

Archive a conversation. Data is preserved, just hidden from main view.

<ParamField path="id" type="string" required>Conversation UUID.</ParamField>

```bash
curl -X POST \
  "https://api.memorybase.app/conversations/a1b2c3d4-e5f6-7890-abcd-ef1234567890/archive" \
  -H "Authorization: Bearer <token>"
```

Returns the updated conversation with `"isArchived": true`.

---

### POST /conversations/:id/unarchive

Restore an archived conversation.

<ParamField path="id" type="string" required>Conversation UUID.</ParamField>

```bash
curl -X POST \
  "https://api.memorybase.app/conversations/a1b2c3d4-e5f6-7890-abcd-ef1234567890/unarchive" \
  -H "Authorization: Bearer <token>"
```

Returns the updated conversation with `"isArchived": false`.
