---
title: "Conversations — Memorybase API Reference"
description: "List, retrieve, and manage AI conversations captured by the Memorybase extension, including messages, project links, and archive state."
---

Conversations are the core resource in Memorybase. Each conversation represents a thread captured from a supported AI platform (such as Claude, ChatGPT, or Gemini). The API lets you list and retrieve conversations, page through their messages, link them to projects, and archive or restore them.

---

### GET /conversations

List all conversations for your account, ordered by most recently updated.

**Query parameters**

<ParamField query="ids" type="string">
  A comma-separated list of up to 100 conversation UUIDs to fetch. Example: `a1b2c3d4-e5f6-7890-abcd-ef1234567890,b2c3d4e5-f6a7-8901-bcde-f12345678901`
</ParamField>

<ParamField query="platform" type="string">
  Filter conversations to a specific platform. Common values: `claude`, `chatgpt`, `gemini`.
</ParamField>

<ParamField query="cursor" type="string">
  A UUID from a previous response's `nextCursor` field. Pass this to fetch the next page of results.
</ParamField>

<ParamField query="limit" type="number">
  Maximum number of conversations to return. Must be a positive integer, maximum `100`.
</ParamField>

**Example request**

```bash
curl "https://api.memorybase.app/conversations?platform=claude&limit=20" \
  -H "Authorization: Bearer <token>"
```

**Example response**

```json
{
  "conversations": [
    {
      "id": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
      "platform": "claude",

      "isArchived": false,
      "createdAt": "2026-02-25T09:00:00.000Z",
      "updatedAt": "2026-02-25T09:45:00.000Z",
      "metadata": {
        "tags": ["typescript", "generics"],
        "topics": ["type-safety", "debugging"],
        "summary": "Discussed how to debug complex TypeScript generic constraints.",
        "userTags": [],
        "userTopics": [],
        "userSummary": null,
        "status": "completed",
        "messageCount": 24
      },
      "projects": [
        {
          "id": "d4e5f6a7-b8c9-0123-defa-234567890123",
          "name": "TypeScript Research",
          "description": "Notes on TypeScript generics and type safety",
          "isArchived": false,
          "version": 3,
          "createdAt": "2026-02-20T10:00:00.000Z",
          "updatedAt": "2026-02-25T09:45:00.000Z",
          "addedAt": "2026-02-25T09:45:00.000Z"
        }
      ]
    }
  ],
  "nextCursor": null
}
```

**Response fields**

<ResponseField name="conversations" type="array">
  Array of conversation objects.

  <Expandable title="Conversation object">
    <ResponseField name="id" type="string">UUID of the conversation.</ResponseField>
    <ResponseField name="platform" type="string">Platform the conversation was captured from (e.g. `claude`, `chatgpt`, `gemini`).</ResponseField>
    <ResponseField name="title" type="string | null">Title of the conversation, or `null` if untitled.</ResponseField>
    <ResponseField name="isArchived" type="boolean">Whether the conversation is archived.</ResponseField>
    <ResponseField name="createdAt" type="string">ISO 8601 timestamp when the conversation was created.</ResponseField>
    <ResponseField name="updatedAt" type="string">ISO 8601 timestamp when the conversation was last updated.</ResponseField>
    <ResponseField name="metadata" type="object | null">
      AI-extracted metadata, or `null` if not yet processed.

      <Expandable title="Metadata fields">
        <ResponseField name="tags" type="string[]">AI-generated tags.</ResponseField>
        <ResponseField name="topics" type="string[]">AI-generated topics.</ResponseField>
        <ResponseField name="summary" type="string | null">AI-generated summary.</ResponseField>
        <ResponseField name="userTags" type="string[]">Tags you have manually added.</ResponseField>
        <ResponseField name="userTopics" type="string[]">Topics you have manually added.</ResponseField>
        <ResponseField name="userSummary" type="string | null">Summary you have manually written.</ResponseField>
        <ResponseField name="status" type="string">Processing status, e.g. `completed`.</ResponseField>
        <ResponseField name="messageCount" type="number">Total number of messages in the conversation.</ResponseField>
      </Expandable>
    </ResponseField>
    <ResponseField name="projects" type="array">Projects this conversation is linked to.</ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="nextCursor" type="string | null">
  Pass this value as the `cursor` query parameter to fetch the next page. `null` means you have reached the last page.
</ResponseField>

### GET /conversations/:id

Retrieve a single conversation by its UUID.

**Path parameters**

<ParamField path="id" type="string" required>
  UUID of the conversation.
</ParamField>

**Example request**

```bash
curl "https://api.memorybase.app/conversations/a1b2c3d4-e5f6-7890-abcd-ef1234567890" \
  -H "Authorization: Bearer <token>"
```

**Example response**

```json
{
  "id": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
  "platform": "claude",
  "title": "Debugging TypeScript generics",
  "isArchived": false,
  "createdAt": "2026-02-25T09:00:00.000Z",
  "updatedAt": "2026-02-25T09:45:00.000Z",
  "metadata": {
    "tags": ["typescript", "generics"],
    "topics": ["type-safety", "debugging"],
    "summary": "Discussed how to debug complex TypeScript generic constraints.",
    "userTags": [],
    "userTopics": [],
    "userSummary": null,
    "status": "completed",
    "messageCount": 24
  },
  "projects": []
}
```

Returns `404` if the conversation does not exist or does not belong to you.

### GET /conversations/:id/messages

List the messages in a conversation, ordered chronologically.

**Path parameters**

<ParamField path="id" type="string" required>
  UUID of the conversation.
</ParamField>

**Query parameters**

<ParamField query="cursor" type="string">
  UUID from a previous response's `nextCursor` field for pagination.
</ParamField>

<ParamField query="limit" type="number">
  Maximum number of messages to return. Must be a positive integer, maximum `100`.
</ParamField>

**Example request**

```bash
curl "https://api.memorybase.app/conversations/a1b2c3d4-e5f6-7890-abcd-ef1234567890/messages?limit=50" \
  -H "Authorization: Bearer <token>"
```

**Example response**

```json
{
  "messages": [
    {
      "id": "c3d4e5f6-a7b8-9012-cdef-123456789012",
      "conversationId": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
      "role": "user",
      "content": "How do I properly constrain TypeScript generic types?",
      "timestamp": "2026-02-25T09:00:00.000Z",
      "createdAt": "2026-02-25T09:00:00.000Z"
    }
  ],
  "nextCursor": null
}
```

**Response fields**

<ResponseField name="messages" type="array">
  <Expandable title="Message object">
    <ResponseField name="id" type="string">UUID of the message.</ResponseField>
    <ResponseField name="conversationId" type="string">UUID of the parent conversation.</ResponseField>
    <ResponseField name="role" type="string">Who sent the message — typically `user` or `assistant`.</ResponseField>
    <ResponseField name="content" type="string">Text content of the message.</ResponseField>
    <ResponseField name="timestamp" type="string | null">Original timestamp of the message on the source platform, or `null`.</ResponseField>
    <ResponseField name="createdAt" type="string">ISO 8601 timestamp when this record was created in Memorybase.</ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="nextCursor" type="string | null">
  Cursor for the next page, or `null` if this is the last page.
</ResponseField>

### POST /conversations/:id/projects

Set the projects associated with a conversation. This **replaces** the full set of linked projects — it is not additive.

**Path parameters**

<ParamField path="id" type="string" required>
  UUID of the conversation.
</ParamField>

**Body**

<ParamField body="projectIds" type="string[]" required>
  Array of project UUIDs to link to this conversation. Maximum 100 items. Pass an empty array to remove all project links.
</ParamField>

**Example request**

```bash
curl -X POST "https://api.memorybase.app/conversations/a1b2c3d4-e5f6-7890-abcd-ef1234567890/projects" \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{"projectIds": ["d4e5f6a7-b8c9-0123-defa-234567890123"]}'
```

**Example response**

```json
{ "success": true }
```

Returns `404` if the conversation or any of the specified projects are not found or are not owned by you.

### DELETE /conversations/:id/projects/:projectId

Remove a single project link from a conversation without affecting other linked projects.

**Path parameters**

<ParamField path="id" type="string" required>
  UUID of the conversation.
</ParamField>

<ParamField path="projectId" type="string" required>
  UUID of the project to unlink.
</ParamField>

**Example request**

```bash
curl -X DELETE \
  "https://api.memorybase.app/conversations/a1b2c3d4-e5f6-7890-abcd-ef1234567890/projects/d4e5f6a7-b8c9-0123-defa-234567890123" \
  -H "Authorization: Bearer <token>"
```

Returns `204 No Content` on success. Returns `404` if the conversation or project link is not found.

---

### POST /conversations/:id/archive

Archive a conversation so it no longer appears in your main view. The conversation and its messages are not deleted.

**Path parameters**

<ParamField path="id" type="string" required>
  UUID of the conversation to archive.
</ParamField>

**Example request**

```bash
curl -X POST \
  "https://api.memorybase.app/conversations/a1b2c3d4-e5f6-7890-abcd-ef1234567890/archive" \
  -H "Authorization: Bearer <token>"
```

**Example response**

Returns the updated conversation object with `"isArchived": true`.

---

### POST /conversations/:id/unarchive

Restore an archived conversation back to your main view.

**Path parameters**

<ParamField path="id" type="string" required>
  UUID of the conversation to unarchive.
</ParamField>

**Example request**

```bash
curl -X POST \
  "https://api.memorybase.app/conversations/a1b2c3d4-e5f6-7890-abcd-ef1234567890/unarchive" \
  -H "Authorization: Bearer <token>"
```

**Example response**

Returns the updated conversation object with `"isArchived": false`.
