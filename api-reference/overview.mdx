## Base URL

```
https://api.memorybase.app
```

## Authentication

Every request requires a bearer token:

```bash
Authorization: Bearer <your-token>
```

See [Authentication](/api-reference/authentication) for token acquisition.

## Request format

All bodies must be `Content-Type: application/json`. All responses return JSON unless noted otherwise.

## Pagination

Cursor-based. Pass the `nextCursor` value from a response as the `cursor` query parameter. `null` means no more results.

```bash
curl "https://api.memorybase.app/conversations?limit=20&cursor=<nextCursor>" \
  -H "Authorization: Bearer <token>"
```

## Errors

```json
{ "error": "Not found" }
```

Some resources use an extended form:

```json
{ "error": { "message": "Project not found", "status": 404 } }
```

| Status | Meaning |
|--------|---------|
| `400` | Bad request |
| `401` | Unauthorized |
| `404` | Not found |
| `413` | Payload too large (2 MB limit) |
| `429` | Rate limited — back off and retry |
| `500` | Server error |
